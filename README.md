# Prompt Shop

AI-powered image analysis and prompt generation web application.

## Features

- Image upload and analysis
- AI-generated prompts  
- User authentication with Google OAuth
- Demo mode support
- Responsive design

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Supabase
- Vercel
